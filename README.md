# Hypersync
Distriubuted p2p file syncing built on [Hyperdrive](https://github.com/mafintosh/hyperdrive)

## Usage
```
Usage: hsync [options]

Options:

   -h, --help              output usage information
   -V, --version           output the version number
   -d, --directory [path]  Sync directory
```
